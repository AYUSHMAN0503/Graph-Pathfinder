# Graph-PathFinder

A beginner-friendly visualization of Dijkstra and A* pathfinding algorithms built with Next.js, TypeScript, and Tailwind CSS.

## Features
- **Algorithms:** Dijkstra (Weighted), A* (Weighted + Heuristic).
- **Interactive:** Draw walls, add weights, move start/end nodes.
- **Controls:** Adjust speed, clear grid, import/export layouts.
- **Education:** Visualizes how algorithms explore graphs.

## How to Run
1. Clone the repo.
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Open `http://localhost:3000`

## Controls Guide
- **Click:** Draw Wall (Black)
- **Shift + Click:** Set Start Node (Green)
- **Alt + Click:** Set Target Node (Red)
- **Ctrl/Meta + Click:** Add Weight (Orange) - *Weights cost 5x more to traverse.*
